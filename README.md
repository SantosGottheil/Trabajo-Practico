# 1.Maquetación UI Clon Whatsapp

### 2.El proyecto trata sobre un clon de un chat de Whatsapp, se reproduce en el trabajo una lista de conversaciones en un sidebar y una conversacion abierta en un panel principal. Se utilizó HTML y CSS únicamente.

### 3.Los archivos del repositorio final son: index.html, styles.css y README.md.

### 4.Para visualizar el trabajo, hay que descargar el repositorio y luego abrir el archivo index.html.

### 5. Para la elección de colores y tipografía fue guía el proyecto base del profesor, salvo casos específicos donde se cambió por otros colores. Se utlizó grid para el contenedor principal y flex en los distintos divs a lo largo del programa.
